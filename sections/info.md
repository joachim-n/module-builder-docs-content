## Info form

The Info tab is the form initially used to create the module. It holds data that
goes in the module's `info.yml` file.

It also allows you to specify where the module will be written. By default,
modules are written to `modules/custom`, but you can specify that the module is
a submodule inside another module, or a test module in another module's test
code.

For more information on the info properties, see [the info file
documentation](https://www.drupal.org/docs/develop/creating-modules/let-drupal-know-about-your-module-with-an-infoyml-file).
